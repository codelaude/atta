# Context Directory

This directory holds auto-generated context files that help agents understand recent project activity.

## Files

| File | Generated By | Purpose |
|------|-------------|---------|
| `recent.md` | `scripts/generate-context.sh` | Last 5 session summaries |

## How It Works

1. Skills run `generate-context.sh` during their finalization step (after session cleanup)
2. The script reads the last 5 session JSON files from `{claudeDir}/.sessions/`
3. It produces `recent.md` with one-line summaries per session
4. Agents read `recent.md` only when user requests session continuity (opt-in, not auto-injected)

## Important

- **`recent.md` is generated** — do not edit it manually (changes will be overwritten)
- **`recent.md` is gitignored** — it contains project-specific runtime data
- **This README is committed** — it documents the directory structure

## Format

`recent.md` contains simple bullet points:

```markdown
# Recent Work Context

*Auto-generated from last 5 sessions. Do not edit manually.*

- **2026-02-17 14:30** — `/tutorial` (completed, 4m 30s) — Agents: project-owner, fe-team-lead
- **2026-02-17 10:15** — `/review` (completed, 2m 15s) — Agents: code-reviewer
```
