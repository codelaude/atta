# Coordinators

> This directory contains team lead agents auto-generated by the `/init` skill.

## How It Works

Run `/init` in your project to automatically detect your tech stack and generate coordinators here. For example:

- If frontend is detected: `fe-team-lead.md` is created
- If backend is detected: `be-team-lead.md` is created
- If both are detected: both coordinators are created

## What Coordinators Do

Coordinators decompose features into specialist tasks and manage their domain:

- **fe-team-lead** — Coordinates frontend specialists (framework, styling, testing, accessibility)
- **be-team-lead** — Coordinates backend specialists (language, framework, database, testing)

## Templates

Coordinators are generated from templates in `.claude/bootstrap/templates/agents/`:
- `fe-team-lead.template.md`
- `be-team-lead.template.md`

## Manual Creation

You can also create coordinator agents manually. Follow the structure of existing coordinator templates and place the file here with a descriptive ID (e.g., `mobile-team-lead.md`).
