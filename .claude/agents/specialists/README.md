# Specialists

> This directory contains technology-specific specialist agents auto-generated by the `/atta` skill.

## How It Works

Run `/atta` in your project to automatically detect your tech stack and generate specialists here. For example:

- If Vue is detected: `vue.md` is created (from `framework-specialist` template)
- If Python is detected: `python.md` is created (from `language-specialist` template)
- If PostgreSQL is detected: `postgresql.md` is created (from `database-specialist` template)
- If SCSS is detected: `scss.md` is created (from `styling-specialist` template)
- If Jest is detected: `jest.md` is created (from `testing-specialist` template)

## Templates

Specialists are generated from universal templates in `.claude/bootstrap/templates/agents/`:
- `framework-specialist.template.md` — For frameworks (Vue, React, Django, Spring Boot, etc.)
- `language-specialist.template.md` — For languages (TypeScript, Python, Java, Go, etc.)
- `database-specialist.template.md` — For databases (PostgreSQL, MongoDB, Redis, etc.)
- `styling-specialist.template.md` — For styling systems (SCSS, Tailwind, CSS-in-JS, etc.)
- `testing-specialist.template.md` — For testing frameworks (Jest, pytest, JUnit, etc.)
- `accessibility-specialist.template.md` — For WCAG/ARIA compliance

## Manual Creation

You can also create specialist agents manually. Place the file here with the technology ID as the filename (e.g., `graphql.md`). The `/agent` skill will find it automatically.
