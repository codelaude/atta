# Skill Templates

> This directory contains templates for project-specific skills generated by the `/init` skill.

## How It Works

During `/init`, detected technologies are mapped to skill templates. The templates use variable substitution to create project-specific skills.

## Template Format

Skill templates use the same variable substitution as agent templates:

```markdown
---
name: {{SKILL_ID}}
description: {{DESCRIPTION}}
---

[Skill instructions with {{VARIABLES}} substituted from detection]
```

## Expected Templates

- `test.template.md` — Test runner skill (adapts to Jest, pytest, JUnit, etc.)
- `build.template.md` — Build skill (adapts to Vite, Webpack, Gradle, etc.)

## Adding New Templates

Create a `.template.md` file here and add a corresponding mapping in `.claude/bootstrap/mappings/skill-mappings.yaml`.
