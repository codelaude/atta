# Generated Skills

> This directory contains project-specific skills auto-generated by the `/atta` skill.

## How It Works

Run `/atta` in your project to automatically detect your tech stack and generate project-specific skills here. For example:

- If a test framework is detected: a `test/SKILL.md` skill is created with the correct test command
- If a build tool is detected: a `build/SKILL.md` skill is created with the correct build command

## Templates

Generated skills come from templates in `.claude/bootstrap/templates/skills/`.

## Manual Creation

You can also create custom skills manually. Each skill needs its own directory with a `SKILL.md` file:

```
generated/
├── my-skill/
│   └── SKILL.md
```

These will be available as `/my-skill` in Claude Code.
